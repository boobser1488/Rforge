
# Forge Interpreter (Rust)

Асинхронный интерпретатор языка **Forge**, написанный на Rust.\
Поддерживает функции (в том числе async), условия, циклы, классы,
обработку ошибок, импорт DLL и встроенные функции.

# Описание проекта

Forge --- интерпретируемый язык с Python-подобным синтаксисом.

Интерпретатор: 1. Читает `.forge` файл 2. Разбирает его в AST 3.
Выполняет код в асинхронной среде 4. Поддерживает встроенные функции и
внешние DLL

Используется `tokio` runtime (`current_thread`).

# Архитектура

## main.rs

Точка входа. Читает файл и запускает интерпретацию.

## parser.rs

Парсинг исходного кода в AST с использованием regex.

## ast.rs

Описание узлов синтаксического дерева.

## eval.rs

Исполнение AST (асинхронное).

## env.rs

Среда выполнения (переменные, функции, классы).

## value.rs

Типы значений языка Forge.

## builtins.rs

Регистрация встроенных функций.
